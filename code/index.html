我叫黄佳欣，27岁，坐标深圳，立志于做一名前端工程师。虽然我并不是计算机专业出生，但是曾有过一年多的UI设计经验， 对网页界面和APP界面都比较熟悉。工作中也涉及到过前台页面的简单书写，后来因为比较感兴趣，于是进行了系统的前端学习。
